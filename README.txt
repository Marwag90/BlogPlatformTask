How to run application?

using VSCode
1- Install dotnet core 8.0.1 and dotnet runtime
2- Install SQL Server 2022
3- Open Project in VSCode
4- From terminal apply migration to create database using command --> dotnet ef database update
5- To run application use command --> dotnet run

using Visual Studio 2022
1- Install dotnet core and c# latest version
2- Install SQL Server 2022
3- Open Project in Visual Studio 2022
4- From terminal apply migration to create database using command --> update-database
5- To run application press ctrl+f5 or run button from the top
